# Patterns

Realizado por: Juan Felipe Mejía Efeer - Susana Pérez Perdomo - Angel David Gomez Sanchez

Problemas y patrones implementados:

Mp3: Se implementa un reproductor mp3 con capacidad de integrar canciones y navegar a traves de las mismas. - Iterator

Barbie: Se plantea un ejemplo de estilizacion de muñecas Barbie en el cual para un determinado escenario se le implementan
un elemento en especifico pero que no altera su funcionalidad - Decorator

Concesionario: Se plantea un ejemplo de clonacion en un ambito en el que se producen en masa vehiculos - Prototype
